
# E-commerce Website
---
## Description
A frontend e-commerce application that allows users to browse products, add items to cart, and simulate ordering.
---
## Tech Stack
- React.js
- JavaScript
- HTML
- CSS
---
## Features
- Product listing
- Add to cart functionality
- User authentication (basic)
- Order simulation with alerts
---
## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
